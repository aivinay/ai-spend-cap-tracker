# Contributing

To add or correct an entry, open an issue or pull request with:
- **Company**, the specific action (cap / license cancellation / centralization), the **figure**, the **date**, and the **tool(s)**.
- **At least one named, linkable source** (reputable outlet or a first-party statement).

Rules:
- **No entry in the main table without a solid source.** Unconfirmed leads go under "Candidate entries — unverified," marked [VERIFY].
- Describe the **mechanism precisely** (e.g., a per-tool dollar cap is different from cancelling a license or centralizing access).
- Keep it **neutral and factual** — this is a reference, not commentary.
